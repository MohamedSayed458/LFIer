# LFIer
Simple Python tool automates the exploitation of Local file inclusion vulnerability

## Usage
```
python3 LFIer.py -u <url> -p <parameter>
```
## Example
```
python3 LFIer.py -u http://testphp.vulnweb.com/showimage.php -p file
```
